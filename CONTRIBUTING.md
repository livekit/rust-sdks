## Contributing to this project

Here are a few guidelines to help you:

### I have found a bug or would like to suggest a new feature

You can do so by [opening a new issue](https://github.com/livekit/rust-sdks/issues/new/choose).
Before you do, please [search existing issues](https://github.com/livekit/rust-sdks/issues) to see if your issue was already reported.

### I have found what I think is a security vulnerability in your code

If it's a sensitive issue, [report the vulnerability privately](https://github.com/livekit/rust-sdks/security/advisories/new).

### I would like to contribute some code to the project

If you're an external contributor, please [fork this repository](https://github.com/livekit/rust-sdks/fork), implement the change and then [create a Pull Request](https://github.com/livekit/rust-sdks/compare). No forking is required, if you have enough access to push your local branch (e.g., if you work at LiveKit).
